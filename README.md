# DevelByte

Personal blog and technical writing by Narendra Kumar.

## Live Sites

- **Main Blog (Docusaurus)**: [develbyte.github.io](https://develbyte.github.io)
- **Terminal Blog**: Minimal terminal-themed design at root

## Terminal Blog

A minimal, terminal-themed blog design optimized for readability and technical content. Built for GitHub Pages with zero build dependencies.

### Features

- Terminal aesthetic with RubiFont ASCII art
- Interactive search with real-time filtering  
- Prism.js syntax highlighting for 16+ languages
- Social media-style code blocks with macOS window buttons
- Zero build step - pure HTML/CSS/JS

### Quick Start

Open `index.html` in your browser or deploy to GitHub Pages.

See `CLAUDE.md` for technical documentation.

## Docusaurus Blog

The main blog is built with Docusaurus and contains posts about distributed systems, databases, and machine learning.

### Development

```bash
npm install
npm start
```

See the Docusaurus documentation for more details.
